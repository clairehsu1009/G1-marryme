# Marry Me

marryme.ddns.net/marryme/index

## 簡介

_結婚一條龍服務網站_

不用找多個網站搜尋，可一次在我們網站上尋找有關結婚的內容，如: 婚宴場地、喜餅、婚戒...等，讓現在想結婚卻又求於效率快的伴侶們，省下許多時間來讓婚禮更完整

![網站地圖](https://github.com/clairehsu1009/G1-marryme/blob/main/img/image.png)

## 功能

1. 註冊
1. 登入、登出
1. 婚宴場地
1. 方案商品
1. 預約賞廳
1. 商品
